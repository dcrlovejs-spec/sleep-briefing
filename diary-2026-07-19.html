WAIC 2026 第二天发布潮：商汤 U1 Pro + 荣耀 Robot Phone + 腾讯具身智能全栈

📚 1 分钟背景

📊 看点

  • 商汤 U1 Pro：从"单点内容生成" 升级到"系统级内容交付"——多模态 AI 的"长程演进" 范式
  • 荣耀 Robot Phone：行业首款机器人手机 + 4DoF 云台系统 + MagicOS Agentic OS，"AI 即类人生命体" 叙事落地
  • 腾讯 ADP 4.0：从消费级智能体到企业级智能体的产品矩阵补齐
  • 60 台人形机器人规模化部署：从"展品" 升级到"公共服务设施"
  • 本届 WAIC 1100+ 家企业参展，3000+ 项展品，300+ 款全球首发

💡 为什么关注

昨天的 WAIC 不只是"又一场 AI 发布会"——而是**中国 AI 头部厂商第一次以"全栈 + 真机 + 工程级证据" 集体亮相**。商汤 U1 Pro 证明"模型层可以交付级"；荣耀 Robot Phone 证明"终端可以具身化"；腾讯 ADP 4.0 证明"应用层可以企业级"——三件事叠加，意味着 2026 下半年"国产 AI 全栈" 叙事第一次有了工程级证据，不再是 PPT。 对投资视角而言，AI 估值锚开始从"算力卡数 / 模型参数" 升级到"全栈工程能力"——单一环节的领先会被全栈厂商稀释，**真正的赢家是能把"芯片 + 模型 + OS + 终端 + 应用" 五位一体打通的厂商**。

🔮 未来影响

短期：WAIC 现场发布的"全栈清单" 将被视为下半年国产 AI 概念股的"弹药库"，国产 AI 板块（芯片 + 模型 + OS + 终端）值得整体关注。中期：随着国产芯片 + 模型 + 终端形成工程闭环，AI 产业的国产化率将显著提升，海外 GPU 与 API 的依赖度持续下降。长期：当"AI 主权" 从概念变成产业链事实，全球 AI 监管框架的博弈点将从"模型对齐" 延伸到"算力主权" 与"数据主权"。

📌 对谁重要：👨‍💻 开发者：Agent OS、国产 GPU 编程模型、具身智能数据平台是下半年的技术热点 💼 投资人：AI 全栈产业链（芯片 / 模型 / OS / 终端 / 数据）值得整体布局 🛒 普通用户：未来 6-12 个月会看到一波"AI 智能体终端" 集中上市，从手机到家电到汽车都会接入

▸ 实用建议

  • 关注 WAIC 官方公众号的"首发产品名单"，锁定 2-3 个你最感兴趣的品类
  • 关注华为昇腾开放计划与阶步 Agent OS 文档，国产 AI 工程栈入门成本在下降
  • 关注"妈祖" AI 预警方案的落地节奏，气象 + AI 是确定性赛道

东方算芯 DF1000：首颗软件定义近存计算 3D AI 芯片，14nm+ 等同 4nm 性能

📚 1 分钟背景

📊 看点

  • 全球首颗软件定义近存计算 3D AI 芯片——架构创新而非工艺追赶
  • 14nm+ 制程 + 3D 垂直封装 = 等同 4nm 性能：国产替代的"第二条路"
  • 520 TFLOPS 单芯片算力 + 128 卡集群验证：已具备工程化部署能力
  • 成熟制程 + 先进封装：规避 EUV 光刻机卡脖子，又保住了性能上限
  • 东方算芯同步展示"超节点、十万卡集群" 的算力底座路线图

💡 为什么关注

DF1000 的真正意义不是"又一颗国产 AI 芯片"，而是**为"国产算力" 找到了"成熟制程 + 先进封装" 这条第二条路**——当 EUV 光刻机被卡脖子、3nm/2nm 工艺追赶难度极大时，通过 3D 堆叠与近存计算架构创新，**14nm+ 也能对标 4nm 性能**。这条路径绕开了"工艺制程" 的硬约束，是中国半导体产业的战略性突破。 从产业格局看，3D 堆叠 + Chiplet + 近存计算的组合，会让国产芯片厂商重新洗牌——拥有先进封装能力的厂商（长电、通富、华天）将获得超额价值。

🔮 未来影响

短期：DF1000 的展出将催化半导体材料、设备、先进封装三个子板块的市场情绪。中期：随着 3D 堆叠技术成熟，国产 AI 算力底座将从"追赶制程" 转向"架构创新"，对 EUV 光刻机的依赖度显著下降。长期：当 Chiplet + 3D 堆叠 + 近存计算形成生态，**"中国半导体路径" 可能成为全球半导体产业的另一极**——与"美国主导的工艺制程路径" 形成结构性互补与竞争。

📌 对谁重要：👨‍💻 开发者：3D 堆叠、Chiplet、近存计算、HBM 高带宽存储是下半年的硬科技热点 💼 投资人：先进封装（长电 / 通富 / 华天）、半导体设备、3D IC 设计工具值得布局 🛒 普通用户：未来 1-2 年 AI 手机、AI PC、AI 汽车的算力成本会因此下降

▸ 实用建议

  • 关注东方算芯后续产品路线图（DF2000 / DF3000）以及 128 卡集群的客户落地
  • 半导体从业者：跨学科补"先进封装 + 架构创新" 知识
  • 关注 TSMC、英特尔的玻璃基板 TGV 进展——3D 封装赛道已是全球共识

微软 7 月安全更新 622 个漏洞创历史新高，BitLocker + SharePoint 高危

📚 1 分钟背景

📊 看点

  • 622 个漏洞单月修复量创历史新高，是 Windows 单月 416 个补丁同比增长 200%
  • BitLocker 加密绕过（CVE-2026-50661）：物理访问可读取加密数据
  • SharePoint Server RCE（CVE-2026-56164）：权限提升后可直接控制服务器
  • 印度最大核电站 Kudankulam 敏感文件遭勒索软件 World Leaks 公开泄露
  • Check Point VPN IKEv1 认证绕过（CVSS 9.3）：远程攻击者可绕过认证

💡 为什么关注

本月补丁数量"史诗级跳涨" 反映的不是 Windows 系统"突然变脆弱"，而是**AI 漏洞挖掘工具大规模应用 + 攻击产业链工业化** 的双重结果。**"漏洞披露到利用" 的窗口期被压缩到 24 小时以内**，传统月度补丁节奏已彻底失效——企业 SOC（安全运营中心）必须从"月度漏洞管理" 转向"实时威胁响应"。 同时，**AI 自身进入攻击载体列表**：Microsoft Teams 必须新增"阻断未授权 AI Bot" 控件、Anthropic Claude 因网络安全告警曾被限制后又解除——AI 既是防御工具也是攻击入口，"AI for Security" 与 "Security for AI" 成为同一议题的两面。

🔮 未来影响

未来 6-12 个月，企业从"采购更多安全产品" 转向"重构身份 / 信任架构"——持续身份验证、设备态势、AI 行为检测会变成默认组件。MFA 已经不算强信号，passkey / 设备绑定 / 行为模式分析将成为新标配。同时 AI 红队（AI Red Team）、AI 漏洞挖掘工具将被大型企业纳入安全运营体系。

📌 对谁重要：👨‍💻 开发者：身份安全、行为分析、AI for SecOps、AI Red Team 岗位需求剧增 💼 投资人：身份安全、AI for Cybersecurity、隐私计算是下半年确定性赛道 🛒 普通用户：警惕"扫码登录" + 短信验证码钓鱼组合，转向 passkey / 密钥登录

▸ 实用建议

  • 个人用户：把重要账号（银行 / 邮箱 / 密码管理器）切换到 passkey 或硬件密钥
  • 开发者：使用 IBAC / RBAC / 零信任等身份治理框架替代传统 MFA
  • 企业：建立"披露-利用 24h 窗口" 的应急响应流程
  • 关注 Adobe / Microsoft / Citrix 的紧急补丁状态

自动驾驶商业化加速：小马智行第七代 Robotaxi 广州深圳投运 + 滴滴 R2 全球化测试

📚 1 分钟背景

📊 看点

  • 小马智行第七代 Robotaxi 广州深圳正式投运：进入"万台量产 + 多城运营" 新阶段
  • 第四代自动驾驶卡车家族 2026 年量产：千台量级，BOM 成本降 70%
  • 滴滴 R2 全球测试 + L4 全栈自主可控：与广汽埃安联合打造
  • 北京亦庄成 BBC 报道焦点：中国无人驾驶已从"测试区" 走向"日常接单"
  • "100% 车规级零部件 + Robotaxi 套件复用"：自动驾驶成本曲线快速下降

💡 为什么关注

自动驾驶本周新闻的真正信号是**"商业化拐点已到"**——小马智行第七代 Robotaxi 投运 + 第四代卡车 2026 年量产 + BOM 成本下降 70%，**意味着自动驾驶从"烧钱测试" 进入"自我造血" 阶段**。当单车成本足够低 + 限定区域法规足够成熟，自动驾驶出租车的盈利模型可以跑通。 对全球格局而言，**中国自动驾驶已从"追赶者" 变成"示范者"**——BBC 蹲点北京亦庄的报道代表西方主流媒体首次承认中国自动驾驶的领先地位。

🔮 未来影响

短期：广州、深圳、北京、上海等一线城市的 Robotaxi 覆盖范围将持续扩大，更多用户能体验。中期：当 Robotaxi 跑通"万台规模 + 单车盈利" 模型，自动驾驶卡车、干线物流、最后一公里配送将同步进入规模化阶段。长期：当 L4 自动驾驶成本低于人类司机，出行市场（出租车 + 网约车 + 物流）将经历结构性重构——出行服务提供商的角色从"司机中介" 转向"车队运营商"。

📌 对谁重要：👨‍💻 开发者：自动驾驶算法、车规级硬件、车队管理平台是下半年热点 💼 投资人：Robotaxi 运营商、激光雷达、车规级芯片、域控制器值得布局 🛒 普通用户：可主动体验广州 / 深圳的 Robotaxi 服务，了解未来出行形态

▸ 实用建议

  • 用户：下载"小马智行" / "萝卜快跑" / "滴滴自动驾驶" 等 App，提前体验
  • 关注 SAE J3069 / 国内 L4 法规细则
  • 自动驾驶从业者：跨学科补"ROS 2 + 机器学习 + 汽车工程" 基础

🛠 ClawHub Top 1：自我改进 Agent / Summarize / Find Skills 等必装技能精选

📚 1 分钟背景

📊 看点

  • 总注册 13729+，5494+ 优质技能，32 个主类别，热榜：编程代理 1222 / Web 开发 938 / DevOps 408
  • 安装命令：`npm i -g clawh` + `clawhub install `，已与 npm 打通
  • 安全机制：每个 skill 都有 VirusTotal 扫描标签（绿色 = Benign）
  • 2026.5.2 Beta 还加了"ClawHub / npm 插件切换" 功能，来源更透明
  • 推荐顺序分四层：生存 / 效率 / 安全 / 进阶
  • 重点推荐覆盖：日历、微信通知、表格自动化、浏览器自动化、AI for Research

💡 为什么关注

从 13000+ skill 池里精选 10 个必装，最大的实用价值是**反选**——避免被 skill 海洋淹没。warm3snow 的分四层（生存 / 效率 / 安全 / 进阶）其实就是一条从"让 OpenClaw 能用" 到"让它真强" 的最小集曲线。 其中 **self-improving-agent** 和 **Proactive Agent** 是两个值得重点关注的方向——前者让 Agent 自我反思迭代，后者让 Agent 从"被动响应" 升级到"主动提醒"。这两类 skill 决定了未来 Agent 是"工具" 还是"伙伴"。

🔮 未来影响

短期：ClawHub 的"必装清单" 类内容会越来越多，本质是**技能市场走向成熟、内容生产者主动精选** 的标志。中期：随着 v2026.5.2 把 ClawHub / npm 切换打通，skill 与 npm 包的界限会进一步模糊。长期：skill 市场可能演化成"私人技能分店"——类似苹果"Apple Intelligence Skill Store"。

📌 对谁重要：👨‍💻 开发者：装 10 个里至少 3 个是你生产效率杠杆点 💼 投资人：ClawHub + 生态相关的上下游（语义搜索、安全扫描、技能管理）值得关注 🛒 普通用户：装一套能让 OpenClaw 替代 80% 日常数字化工作的最小集

▸ 实用建议

  • 先装三层："生存层"（最少配置）→"效率层"（明显节省时间）→"安全层"（防止数据外泄）

  • 重点体验 self-improving-agent：让你的 Agent 学会自我反思
